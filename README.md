NFS Garage is a lightweight mod manager for Need for Speed Most Wanted using **ModLoader**.  
It lets you import mods, manage them, and launch the game easily from one simple interface.

---

## ✨ Features

- Import mods (first extract RAR, then open the app)  
- Mod library with enable/disable and delete options  
- Mod history  
- Install ModLoader if needed  
- Launch the game directly  

---

## 🚀 How to Use

1. Extract your mod RAR to a folder  
2. Open NFS Garage
3. Click Import and select the mod folder  
4. Manage your mods in the library (enable, disable, delete)  
5. Launch the game from the app
